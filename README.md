Eigenanteil zu meiner W-Seminararbeit über "Die Mathematik hinter GPS - Positionsbestimmung mit Satelliten mathematisch modelliert und programmiert"
In diesem GitHub Repository bzw. Ordner findet sich der Eigenanteil der Arbeit.
In der Arbeit wird die Programmierung als Quelle [6] bezeichnet.

Öffnen und Ausführen des Programms:
1.  Kopieren des Ordners oder Clonen des Repositories auf Festplatte (bevozugter Weg)
    Ausführen des Programms über Entwicklungsumgebung (z. B. VS Code)
2.  Kopieren des Ordners oder Clonen des Repositories auf Festplatte
    über cmd mit "cd Pfad\kopiertes\Projekt\Berechnung\src\main\java\org\wseminar\eigenanteil" Projektordner öffnen
    mit "java Calculation.java" Programmcode ausführen

Im Terminal wir errechnete sowie umgerechnete Position nach Bespieldaten ausgegeben
in Berechnung\results wird ein neues output.txt File gespeichert
Achtung: Das Programm handelt keine eingabefehler (nur korrekte Eingaben der Daten von vier Satelliten)
